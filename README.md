A lightweight NFT Marketplace smart contract optimized for the **Base L2** network.

## Features
- **Listing**: Put your ERC-721 tokens for sale.
- **Buying**: Purchase NFTs using native ETH on Base.
- **Low Fees**: Built to take advantage of Base's efficiency.

## Deployment (Base Sepolia)
1. Install Foundry or Hardhat.
2. Setup your `.env` with Base RPC URL.
3. Deploy using: `forge create --rpc-url $BASE_RPC ...`
